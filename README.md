# Python-Projects

## About this Repository
___

A variety of Python applications, scripts, and notebooks.
They highlight my experience with data analysis, automation, web scraping, APIs, and problem-solving using Python’s standard library and popular packages.

## Featured Projects
___

* [Promotion Eligibility and Bias Analysis](https://github.com/judeonuh/Promotion-eligibility-and-bias-analysis.git): Employees of a company raised concerns that promotions are skewed and biased. This project uses machine learning to answer the business question: “Is the current promotion process biased? Which features should be used to make fair, explainable promotion eligibility decisions?”
* [Stroke Prediction Model](https://github.com/judeonuh/Stroke-Prediction-Model.git): This project compares 5 models and selects the best model to predict whether a patient is likely to get stroke based on the input parameters like gender, age, various diseases, and smoking status.
* [Task Manager](https://github.com/judeonuh/finalCapstone.git): This program allows a user to manage their tasks. Program has been refactored from an initial simple task management system, incorporating new features.
